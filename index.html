=[ ZorTrack Ver 0.02 ]====================

ZorTrack - Менеджер задач.

Предназначен для управления задачами пользователей. Возможен просмотр статистики за определённые интервалы времени по каждому пользователю.

Разделение прав на обычных пользователей и администраторов. Администраторы могут пользоваться всеми фунциями ZorTrack. Пользователи, могут просматривать свои задачи, закрывать их, создавать для себя задачи. Пользователи не могут просматривать или редактировать других пользователей или удалять задачи созданные не ими.

Для установки необходим веб сервер потдерживающий PHP >5 и MySQLбаза данных.

[УСТАНОВКА]

  • Скопируйте файлы в нужную вам папку на сервере
  • Откройте файл /system/application/config/config.php
    Замените в переменной $config['base_url'] на урл вашего сайта
  • Откройте файл /system/application/config/database.php
    Поменяйте настройки доступа к базе на ваши
  • В вашей базе создайте таблицу с именем которое вы вписали при настройке базы, как указанно выше
  • Сделайте импорт sql файла со структурой таблиц zortrack.sql
  • Зайдите через браузер по адресу который вы прописали для этого менеджера, например http://localhost/zortrack
  • При первом открытии менеджера, когда ещё нет пользователей, вам будет предложено создать первого пользователя, он будет с административными правами.
    Введите логин (логин обязательно должен быть в виде e-mail) и пароль (минимум 4 символа)
  • Можете пользоваться

[НАСТРОЙКА]
Других настроек в менеджере практически нет, разобраться и работать с ним довольно просто.
Если вы захотите сменить оформление, то вам нужно будет менять файлы вида (представления) которые находятся /system/application/views/
CSS, JS скрипты, изображения находятся в корне с именами css, js, images
Практически всё оформление можно настроить меняя css файл /css/ztrack.css

P.S. Использование данного менеджера полностью на ваш страх и риск. Приложение не тестировалось на устойчивость к взломам и писалось для внутренних нужд фирмы. Но если у вас появятся замечания или предложения, то я с радостью выслушаю их.

P.P.S. Написано с использованием CodeIgniter и NetBeans :)

=== English

ZorTrack - Task Manager.
Designed to manage the tasks of users.
You can view statistics for the specified intervals for each user.

The division of rights to ordinary users and administrators.
Administrators can use all the functions being ZorTrack.
Users can view their tasks, close them, to create for themselves the task.
Users can not view or edit or delete other users' tasks were not created by them.

To install the necessary web server PHP > 5 and MySQL base.

[SET]

  • Copy the files to the desired folder on the server
  • Open the file /system/application/config/config.php     Replace the variable $ config ['base_url'] url on your site
  • Open the file /system/application/config/database.php     Change the setting up access to the database on your
  • In your database, create a table with the name that you entered when you set up the base, as above
  • Make the import sql file with the structure of the tables zortrack.sql
  • Go through the browser to the address that you have registered for this manager, for example http://localhost/zortrack
  • When you first open the manager, when there are no users, you will be prompted to create the first user, it will be with administrative rights.     Enter your login (username must be in the form of e-mail) and password (minimum 4 characters)
